# Assets

The source repository (youki0p0/GRID-STELLA) shipped **no binary assets** — no
logos, icons, fonts, or imagery — only a written development spec (`claude.md`).

GRID STELLA's iconography is therefore an **emoji sigil set**, not files:

- Instruments: 📌 観測針 · 🌐 天球儀 · 🧭 羅針盤の刻印 · ⚱️ 均衡の分銅 · 🧥 方位外套
- Glyph-icons (unicode): ◈ (gold) · ⭐ (synergy) · ⚙ (tune) · ↻ (reroll/rotate)
  · ⚔ (attack) · ◔ (cooldown) · ⛨ (shield) · → (advance)

The only generated graphic is a **CSS/SVG geometric astrolabe backdrop** (rings,
ticks, constellation lines) rendered inline in the UI kit — decorative chrome,
not an icon asset.

If/when real brand assets (a wordmark logo, engraved-line instrument sigils,
intended fonts) become available, drop them here:

- `assets/fonts/` — webfonts, wired via `@font-face` in `tokens/fonts.css`.
- `assets/icons/` — bespoke SVG sigils, if emoji are ever replaced.
- `assets/logo/` — wordmark / lockup.

See `readme.md` §4 (Type substitution flag) and §6 (Iconography).
