# 神楽マキナ — 機械神の回路（CANONICAL SPEC）

> このドキュメントが現行の正準仕様。`docs/ARENA.md` の旧設計と差分がある場合は
> **本書を正**とする。テーマは「武器を並べるゲーム」ではなく
> **「機械神の回路を構築するゲーム」**。プレイヤーはコアから電力を引き回し、
> 武器・補助装置を回路として組み、自動戦闘で戦う。

## MVP スコープ

実装する: ショートモード / 3ジョブ / 共通アイテム / ジョブ固有アイテム /
グリッド（コア＋電力接続）/ オートバトル / バフデバフ。
実装しない（後日）: PvP通信 / ランクマッチ / リプレイ / 課金 / シーズン / 追加職業 /
ロングモード。

## 中核システム

### エネルギー（最重要）
- 初期 最大5 / 現在5、毎秒 +1 回復。
- 全武器は **ダメージ / クールダウン / エネルギー消費** を持つ。
- 武器は「CD経過 かつ エネルギー≥消費」で発動し、発動時に消費。
  → 強武器を並べてもエネルギー不足で動かない＝配置と回転率の戦略性。
- 例: レールガン(Dmg50 / CD6s / 消費4)、マシンガン(Dmg5 / CD0.5s / 消費1)。

### グリッドと電力
- **コア**: 初期配置の電力供給源。
- グリッドはコアに接続されていないと機能しない。武器も電力供給が必須。
- グリッド種類:
  - 通常(4マス): 効果なし・拡張用
  - シールド(4): 戦闘開始時シールド付与
  - クロック(2): 配置アイテムの速度上昇
  - バッテリー(3): 最大エネルギー +1
  - パワー(3): クリティカル率上昇

### 状態異常（全ジョブ共通・少数精鋭）
スタック型: 過電圧(被ダメージ+1%/stack) / ウイルス(継続ダメージ) /
ジャミング(命中低下) / フリーズ(速度低下) / メモリリーク(エネルギー妨害)。
非スタック: クラッシュ(対象アイテムを一定時間停止, 例2秒)。

### バフ
強化(武器Dmg+1) / 加速(速度+1%) / シールド / 最大エネルギー / エネルギー回復 /
クリティカル率 / クリティカルダメージ / 命中 / 無敵(一定時間ダメージ無効) /
ファイアウォール(デバフ無効) / トゲ(被弾時固定反射)。

## ジョブ（共通ルールの使い方で差を出す）

状態異常は共通。職業専用の状態異常は作らない。差は **蓄積 / 起爆 / 参照**。

- **リアクターストライカー** — 高火力アタッカー。状態異常を**起爆**。
  得意: 過電圧・クラッシュ。過電圧を盛って高火力で起爆。弱点: 低防御・連撃に弱い。
- **インダクションガンナー** — 高速連射。状態異常を**蓄積**。
  得意: ウイルス・ジャミング。状態異常を高速で積む。弱点: 高耐久。
- **マトリクスキャスター** — 妨害・耐久。状態異常を**参照**。
  得意: フリーズ・メモリリーク。例「対象のフリーズ値×2ダメージ」。弱点: 瞬間火力不足。

三すくみ: キャスター→ガンナー→ストライカー→キャスター（ビルドで覆る）。

## アイテム出現ルール
- 共通アイテム: 全職出現。
- ジョブ固有: 選択職のみ。
- 他職アイテム: 通常出現せず、特殊アイテム経由でのみ取得可能。

## 設計思想
初心者最優先 / 状態異常は少数 / 職業ごとに別ルールを作らない /
「共通ルールをどう使うか」で差を出す / 回路とエネルギーを組む game。
